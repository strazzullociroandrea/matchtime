### Cos'è Match Time?

Match Time è un'**applicazione web (PWA)** che permette la visualizzazione di tutte le partite, gli orari
e i campi di gioco, della propria squadra di pallavolo, in un unica vista organizzata, su qualsiasi dispositivo mobile e non, dotato di una connessione ad internet.

---

### Funzionalità principali

| Servizio                             | Descrizione                                                                                                                        
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| **Visualizzazione partite**          | Visualizza tutte le partite della tua squadra, con orari e campi di gioco.                                                         
| **Dati sempre aggiornati**           | I dati vengono aggiornati ogni 12 ore circa, garantendo informazioni sempre accurate e aggiornate, tramite API appositamente creata.                          
| **Accesso da qualsiasi dispositivo** | Match Time è accessibile da qualsiasi dispositivo con un browser web, offrendo un'esperienza utente fluida e intuitiva.            
| **Link al navigatore**               | Ogni partita include un link al navigatore, facilitando la localizzazione del campo di gioco.                                                                             
| **Sincronizzazione con calendario**  | Permette di sincronizzare le partite con il calendario del dispositivo tramite iscrizione.                                                                                    
| **Installa come app**                | Puoi installare Match Time come un'applicazione nativa sul tuo dispositivo, per un accesso rapido e senza dover aprire il browser. 


---

### Stack tecnologico

- Framework: Astro
- Linguaggio: TypeScript (TSX)
- Style: Tailwind CSS
- Component UI: Shadcn UI
